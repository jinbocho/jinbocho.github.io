Domande frequenti

Jinbocho Community è davvero gratuito?
Sì. La Community Edition è completamente gratuita per uso personale e familiare, senza limiti di libri o utenti. Basta un server Docker.
Cos'è il modulo AI Pro?
Il modulo AI Pro aggiunge due funzionalità alla piattaforma: descrizione automatica dei libri — quando la scansione ISBN non trova una sinossi nei database bibliografici (Open Library, Google Books), un LLM genera automaticamente la descrizione; e riconoscimento di titolo e autore via machine learning — per catalogare libri il cui ISBN non è leggibile o non è presente online. La generazione della descrizione è compatibile con qualsiasi provider LLM (OpenAI, Gemini, Groq, OpenRouter, Ollama). Altre funzionalità AI verranno aggiunte nel tempo.
I miei dati rimangono miei?
Sempre. Jinbocho gira sulla tua infrastruttura, su un tuo database PostgreSQL. Nessun dato viene inviato a server esterni, a meno che tu non configuri un provider LLM (solo per la funzione AI Pro).
Posso provare l'AI prima di acquistare la licenza Pro?
Scrivi a jinbochoapp@gmail.com per concordare una demo o un periodo di valutazione.
Cos'è il piano Assistito?
Il piano Assistito include le stesse funzionalità di Community, più la prima installazione in produzione a cura mia: configurazione Docker Compose oppure deploy cloud su Render + Neon, verifica del funzionamento e test post-deploy. Il costo è di €99 una tantum. L'infrastruttura cloud o il server è a carico dell'utente.
Il costo del server è incluso?
No. In tutti i piani il software è self-hosted: il costo dell'infrastruttura (server, VPS, NAS o cloud provider) è interamente a carico dell'utente. Per un deploy su Render + Neon il costo di partenza può essere zero (free tier) o pochi euro al mese a seconda del traffico. Scrivi a jinbochoapp@gmail.com per una stima personalizzata.
