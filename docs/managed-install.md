# Managed Installation Service

Jinbocho is designed to be self-hosted — your data stays on your infrastructure, always.
If you'd rather skip the technical setup, a managed installation service is available.

## What's included

- Full deployment on your server, VPS, or private cloud (Render, Railway, Hetzner, etc.)
- Production-grade configuration: environment variables, secrets, database setup
- Optional AI module (Pro edition) — LLM provider configuration included
- Post-install verification and handoff documentation

## What's NOT included

- Hosting costs (you pay your own server/cloud provider directly)
- Ongoing maintenance (available as a separate agreement)
