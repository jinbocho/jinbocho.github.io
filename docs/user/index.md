## What is Jinbocho?

Jinbocho is a **home library management system** that helps families keep track of
their physical book collections. Named after the famous book district in Tokyo, it lets you:

- Catalogue every book with title, author, ISBN, and cover
- Know exactly which room and shelf each book is on
- Track reading progress across the whole family
- Scan ISBN barcodes to add books in seconds
- Search your entire collection instantly
- Export your library as CSV or JSON

Jinbocho runs in the browser — no installation needed. It works on phones, tablets,
and computers. Multiple family members can use it simultaneously with different
permission levels.
