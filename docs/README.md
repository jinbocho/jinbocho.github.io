# Jinbocho Manuals

Complete documentation for Jinbocho: Home Library Management System.

## What You'll Find Here

This directory contains two comprehensive manuals:

### 👨‍💻 [Developer Manual](developer/index.md)

For developers, DevOps engineers, and maintainers who need to:
- Set up local development environment
- Deploy Jinbocho to production (Render + Neon)
- Understand system architecture
- Troubleshoot deployment issues

**Key sections**:
- Prerequisites (Docker, Python, Node.js)
- Local development with Docker Compose
- Backend services overview
- CI/CD and GitHub Actions
- Production deployment guide
- Monitoring, logging, troubleshooting

**Time to read**: ~2-3 hours for a complete understanding

---

### 👥 [User Manual](user/index.md)

For end-users and family members who want to:
- Create an account and get started
- Add and organize books
- Search and filter your library
- Invite family members
- Track reading progress

**Key sections**:
- Getting started and account creation
- Authentication and account management
- Adding and managing books
- Organizing locations (rooms, bookcases, shelves)
- Advanced features (scanning, export, bulk operations)
- Family management and user roles
- Mobile tips and localization

**Time to read**: ~1-2 hours to learn all features

---
