# Per iniziare con Jinbocho

Benvenuto in Jinbocho, il sistema di gestione della biblioteca domestica per la tua famiglia. Questa guida ti aiuterà a configurare il tuo account e iniziare a catalogare i libri in pochi minuti.

## Requisiti di sistema

### Browser supportati

Jinbocho funziona su qualsiasi browser moderno:

- **Chrome** (v90+)
- **Firefox** (v88+)
- **Safari** (v14+)
- **Edge** (v90+)

### Supporto mobile

Jinbocho è completamente responsive e funziona su:

- **iPhone/iPad** (iOS 12+, Safari)
- **Smartphone/tablet Android** (Chrome, Firefox)

### Connessione internet

Jinbocho richiede una connessione attiva. Non è disponibile una modalità offline.

## Accedere a Jinbocho

Apri il browser e vai all'URL della tua installazione di Jinbocho. Se è la prima volta, vedrai la schermata di login. Clicca **"Non hai un account? Registrati"** per creare il tuo account famiglia.

## Creare l'account famiglia

### Passo 1: compila i dati

Il modulo di registrazione chiede:

| Campo | Esempio | Note |
|-------|---------|------|
| **Nome famiglia** | "Famiglia Rossi" | Visibile a tutti i membri |
| **Il tuo nome** | "Marco" | Il tuo nome personale |
| **Email** | "marco@esempio.it" | Per il login e recupero password |
| **Password** | (nascosta) | Minimo 8 caratteri |

### Passo 2: crea l'account

Clicca **"Registrati"**. L'account viene creato immediatamente.

Cosa succede subito dopo:

- Diventi l'**Admin** della tua famiglia (accesso completo)
- Puoi invitare altri familiari
- La biblioteca è vuota (pronta per aggiungere libri)

### Passo 3: accedi

Dopo la registrazione potrebbe essere richiesto di accedere:

1. Inserisci la tua email
2. Inserisci la password
3. Clicca **"Accedi"**

Sei dentro la tua dashboard Jinbocho.

## Il primo sguardo all'app

### Barra di navigazione

**Su desktop** (barra laterale sinistra):
- **Dashboard** — panoramica della biblioteca
- **Libri** — catalogo completo
- **Posizioni** — stanze e librerie
- **Statistiche** — progressi di lettura
- **Prestiti** — libri in prestito
- **Utenti** — gestione familiari
- **Impostazioni** — preferenze e profilo

**Su mobile** (barra inferiore):
- Icone per le sezioni principali
- Il menu hamburger (☰) in alto apre la navigazione completa

## Aggiungere il primo libro

### Opzione 1: scansione ISBN (più veloce)

1. Clicca **"Aggiungi libro"** → **"Scansiona ISBN"**
2. Consenti l'accesso alla fotocamera quando richiesto
3. Punta la fotocamera sul codice a barre del libro
4. L'app riconosce l'ISBN e cerca i metadati automaticamente
5. Controlla titolo, autore, copertina
6. Scegli dove mettere il libro (stanza → libreria → scaffale)
7. Clicca **"Salva"**

### Opzione 2: inserimento manuale

Se la scansione non funziona:

1. Clicca **"Aggiungi libro"** → **"Inserimento manuale"**
2. Digita titolo, autore, ISBN (opzionale), editore, anno
3. Scegli la posizione nella biblioteca
4. Clicca **"Salva"**

## Organizzare la biblioteca (primo setup)

Prima di aggiungere molti libri, crea le tue posizioni:

### Passo 1: crea una stanza

1. Vai su **Posizioni** (barra laterale)
2. Clicca **"Nuova stanza"**
3. Inserisci il nome: "Salotto", "Camera", "Studio", ecc.
4. Clicca **"Crea"**

### Passo 2: crea una libreria

1. Clicca sulla stanza appena creata
2. Clicca **"Nuova libreria"**
3. Inserisci il nome: "Billy IKEA", "Libreria grande", ecc.
4. Clicca **"Crea"**

### Passo 3: inizia ad aggiungere libri

Ora puoi assegnare ogni libro a uno scaffale preciso.

## Invitare i familiari (opzionale)

Solo l'Admin può invitare nuovi membri.

1. Vai su **Utenti**
2. Clicca **"Invita utente"**
3. Inserisci l'email del familiare
4. Scegli il ruolo:
   - **Admin**: accesso completo (gestione utenti, posizioni, impostazioni)
   - **Editor**: può aggiungere e modificare libri e posizioni
   - **Viewer**: sola lettura (può sfogliare i libri)
5. Clicca **"Invia invito"**

Il familiare riceve un'email con il link di registrazione. Cliccandolo crea il proprio account e viene aggiunto alla biblioteca di famiglia.
